# Project Page of EduBoard
## A Microcontroller-Board based on ATXmega128A3U for Juventus Technikerschule in ZÃ¼rich
---
## Main Features:
- ATXmega128A3U
- onboard JTAG-Debugger
- 4 Buttons
- 8 LED's
- 2 Poti's
- 2 PWM-Outputs
- onboard 9 Axis IMU
- 4x20 Character-Display
- USB-Seraial Connection
- Breakout-Board Section
- X-Bee Connector for Wireless communication
- FreeRTOS Ready [https://github.com/mburger82/FreeRTOS10_Template_Demo]

## Microcontroller Features:
- 128k Flash
- 8k Ram
- 2k Eeprom
- 32MHz RISC CPU 8/16Bit
- Hardware Multiplier
- 2 ADC-Modules
- 2 DAC-Modules
- 7 16Bit Timer
- 7 USART-Interfaces
- 2 I2C-Interfaces
- 3 SPI-Interfaces
- AES & DES Crypto Engine
- CRC
- JTAG-Debugging
